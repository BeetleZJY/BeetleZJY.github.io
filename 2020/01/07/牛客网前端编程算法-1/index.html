前言

这里收集网上其他博主以及牛客网的一些算法编程题，给出网上巧妙的算法以及加上自己的一些备注和自己的做法作为面试的题型的储备

1.斐波那契数列：1、1、2、3、5、8、13、21。输入n，输出数列中第n位数的值

我们通过观察数列的规律，发现当下标i>=1时，当前的值总是等于fn(i-1) - fn(i-2). 比如 2 = 1 - 1，3 = 1 - 2 …
那么根据此规律，我们可以用递归解决。

2.对数组 [15,32,12,5,7,6] 进行排序

  • 1.经典排序

    逻辑：每次比较相邻两个值，如果前面的值大于后面的值，则两个值交换。(内循环)，重复循环比较数组长度次数。(外循环)即可排序得出结果。

  • 2.原生sort方法

3. [5,3,7,8,8,3,10] 数组去重

  • 2.往新数组添加不存在的值

4. [5,3,7,8,8,3,10] 数组重复的元素

  • 1.逻辑: 遍历每一项，新建temp数组保存结果。利用indexOf和lastIndexOf,当两者不相等，并且res数组中没有这个元素的时候，说明这个元素第一次重复，push进结果数组

  • 2.逻辑: 利用排序后的数组重复的元素为相邻的，那么利用filter方法把他们过滤出来

  • 3.逻辑: 利用对象唯一性，循环数组的值，如果数值的值不存在对象中，则存入为对象键， 否则条本次循环并存入重复键的值到另外一个临时数组中
