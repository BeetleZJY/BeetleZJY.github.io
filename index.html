Hexo简介

作为前端技术栈，Hexo是一款基于Node.js的静态博客框架，依赖少易于安装使用，可以方便的生成静态网页托管在GitHub和Coding上，是搭建博客的首选框架。另外该框架对中文支持十分友好， 大家有兴趣可以官网了解下。

Hexo搭建步骤

1.安装git

安装git用于部署代码上传博客静态文件。

git淘宝镜像， 下载速度快， window版本直接安装点击下一步完成安装即可。

linux 安装， 安装完之后输入 git –version 返回版本安装成功。
